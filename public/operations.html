Agent operating procedure

Refund / Return Resolution

v1.0GlobalChat + 3Active mock

What is this AOP used for?

Use this AOP when a customer asks to return a book, request a refund, report a damaged book, report a wrong item, or resolve a return eligibility issue.

Natural-language procedure

  1. Gather required information. Confirm authentication. Ask for the order number and return reason when missing.
  2. Check order ownership and status. Use @orders.lookup with the authenticated session. Reference {{order.delivered_days_ago}}.
  3. Check return eligibility. Use @policy.check_return_window. Continue at 30 days or fewer; otherwise link to Escalation Request AOP.
  4. Classify return reason. Damaged, crushed, broken, torn, or wrong items qualify. Escalate unclear reasons.
  5. Resolve eligible refund. Use @refund.create_mock and return funds to the original payment method.
  6. Escalate if needed. Use @escalation.create_mock and explain that a teammate must review the case.
