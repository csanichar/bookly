Editing AOP

Refund / Return Resolution

v1.0GlobalActive mock

What does this AOP do and when should it be used?

Use this AOP when a customer asks to return a book, request a refund, report a damaged book, or resolve return eligibility.

AOP Logic

Global

Use @ for tools, # for linked AOPs, and {{metadata}} for conversation data.

  1. Confirm {{user.authenticated}} before using private order tools. Ask for the order number and return reason when missing.
  2. Use @orders.lookup with {{user.username}} and {{order.order_number}}. Do not reveal data for another customer.
  3. Use @policy.check_return_window. If {{order.delivered_days_ago}} is more than 30, go to #Escalation Request.
  4. Damaged, crushed, broken, torn, or wrong items qualify for an automatic mocked refund.
  5. Use @refund.create_mock when eligible. Otherwise use @escalation.create_mock.
